# Word2Vec implementation with NumPy
Word2Vec skip-gram model with negative sampling on NumPy

## Launch

1. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
2. Run the app:
    ```bash
    python main.py
    ```

## Dataset

Source: [BBC Full Text Document Classification](https://www.kaggle.com/datasets/alfathterry/bbc-full-text-document-classification)
